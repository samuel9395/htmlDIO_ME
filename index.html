HTML:

HTML é uma linguagem de marcação de textos para a Web. É a linguagem base para a construção de páginas da web, definindo a estrutura e o conteúdo delas.

Estrutura: HTML usa uma série de elementos ou tags para definir a estrutura de uma página. Esses elementos são escritos entre colchetes angulares (< >).
Exemplos de tags incluem < html >, < head >, < body >, < h1 > a < h6 > (para cabeçalhos), < p > (para parágrafos), < a > (para links), < img > (para imagens), e muitas outras.

Falando sobre tags:

< html >: A tag raiz de um documento HTML. Todos os outros elementos devem estar contidos dentro dela.
< Head >: Contém metadados sobre o documento, como seu título, links para estilos e scripts, e meta informações.
< Body >: Contém o conteúdo visível da página, como textos, imagens, vídeos, etc.

Estrutura de Conteúdo:

Cabeçalhos (< h1 > a < h6 >): Usados para definir títulos e subtítulos. < h1 > é o mais importante e < h6 > é o menos importante.
< p >: Define um parágrafo de texto.
< a >: Define um link para outra página ou para um e-mail.
< img >: Define uma imagem.

Estrutura de Listas:

Listas Ordenadas < ol >: Cria uma lista numerada.
Listas Não Ordenadas < ul >: Cria uma lista não numerada.
< li > Itens de lista < li >: Cria um item de lista.
Listas de Definição < dl >: Usadas para definir uma lista de termos e descrições.
< dt >: Define um termo.
< dd >: Define uma descrição.

Estrutura de Formulários:

< form >: Define um formulário HTML para entrada de dados do usuário.
< input >: Define um campo de entrada de dados em um formulário.
< label >: Define um rótulo para um campo de entrada de dados em um formulário
< textarea >: Define uma área de texto multilinhas.
< button >: Define um botão clicável.

Estrutura Semântica:

< header >: Define um cabeçalho para um documento ou seção.
< nav >: Define um conjunto de links de navegação. < nav >< a href="#home">Início< /a>< /nav >
< article >: Define conteúdo independente e autocontido.
< section> : Define uma seção de um documento.
< footer >: Define um rodapé para um documento ou seção. < footer >< p >Rodapé do site.< /p> < /footer >

Atributos:

  1. id: Define um identificador único para a tag, permitindo estilos e scripts serem aplicados especificamente a essa tag.
  2. class: Especifica uma ou mais classes que podem ser aplicadas à tag, permitindo agrupamento e aplicação de estilos em várias tags.
  3. src: Usado em tags como 'img' e 'script' indica o caminho para o recurso que a tag deve carregar
  4. href: Usado em tags como < a > (âncora), indica o destino do link.
  5. alt: Utilizado em tags como < img >, fornece um texto alternativo que é exibido quando a imagem não pode ser carregada ou lida por leitores de tela.
  6. title: Adiciona um texto informativo que é exibido quando o cursor paira sobre a tag.
  7. style: Permite a definição de estilos CSS diretamente na tag.
  8. width e height: Especificam as dimensões da tag, geralmente utilizadas em tags como < img >.
  9. rel: Utilizado em tags como < a > para definir a relação entre o documento atual e o documento vinculado.
  10. type: Especifica o tipo de conteúdo de tags como 'script' e 'style'.

inspetor de elementos

O Inspetor de Elementos é uma ferramenta integrada aos navegadores web modernos que permite aos desenvolvedores e usuários visualizar e interagir com a estrutura HTML, CSS e JavaScript de uma página web em tempo real.

Principais Funcionalidades:

  1. Visualização de HTML e CSS
  2. Edição em Tempo Real
  3. Depuração de JavaScript
  4. Análise de Rede
